> [!warning] Overdue
> ```tasks
> not done
> due before today
> path includes projects/
> group by filename
> sort by due
> ```

---

> [!info] Due today
> ```tasks
> not done
> (due on today) OR (scheduled on today)
> group by filename
> sort by due
> ```

---

### All open project tasks, grouped by project

```tasks
not done
path includes projects/
group by filename
sort by due
```

---

### Recently completed (last 7 days)

```tasks
done after 7 days ago
group by filename
sort by done reverse
```

---
